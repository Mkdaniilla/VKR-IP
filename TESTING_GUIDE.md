## 📋 Тест 2: Страница "Оценка ИС" (Капитализация 2.0)

### Шаги:
1. Перейдите в раздел **"Оценка ИС"** или **"Капитализация 2.0"**
2. Проверьте селектор **"Тип объекта"**:
   - Должны быть видны **4 группы** с эмодзи
   - Каждая группа содержит соответствующие типы

3. Выберите тип **"Программы для ЭВМ"** (software)
4. В поле **"Направленность / Категория"** выберите **"SaaS Решение / Веб-сервис"**
5. Проверьте, что появился блок **"Специфические метрики (saas)"** с 3 слайдерами:
   - Сложность архитектуры / Кода
   - Масштабируемость
   - Зрелость стека

6. Заполните форму:
   - Годовой доход: `1000000`
   - Инвестиции (R&D): `500000`
   - Оставшийся срок действия: `10` лет
   - Сила бренда: `7`
   - Ставка роялти: `5%`

7. Нажмите **"Сформировать отчет"**

### Ожидаемый результат:
✅ Форма заполняется без ошибок  
✅ Специфические метрики отображаются для выбранного типа  
✅ Расчет выполняется успешно  
✅ PDF-отчет генерируется и доступен для скачивания
