# Типы генерируемых документов

## Обзор
Система поддерживает автоматическую генерацию 8 типов юридических документов с использованием AI для создания профессионального контента.

## Доступные типы документов

### 1. ⚖️ Досудебная претензия (pretension)
**Назначение:** Официальное требование о прекращении нарушения прав до обращения в суд  
**Ключевые разделы:**
- Описание нарушения
- Правовое обоснование (статьи ГК РФ)
- Требования к нарушителю

### 2. 🔒 Соглашение о конфиденциальности - NDA (nda)
**Назначение:** Защита конфиденциальной информации об объекте ИС  
**Ключевые разделы:**
- Предмет соглашения
- Определение конфиденциальной информации
- Обязательства по неразглашению
- Ответственность и срок действия (5 лет)

### 3. 💰 Договор отчуждения исключительного права (assignment)
**Назначение:** Полная передача прав на объект ИС от одного лица другому  
**Ключевые разделы:**
- Предмет договора (полное отчуждение)
- Вознаграждение
- Гарантии чистоты прав
- Момент перехода права

### 4. 📄 Лицензионный договор (license)
**Назначение:** Предоставление права использования объекта ИС с сохранением прав правообладателя  
**Ключевые разделы:**
- Способы использования (scope)
- Территория использования
- Срок действия лицензии
- Условия эксклюзивности

### 5. 🏛 Исковое заявление (isk)
**Назначение:** Судебная защита нарушенных прав на объект ИС  
**Ключевые разделы:**
- Исковые требования
- Фактические обстоятельства
- Правовое обоснование (ст. 1252 ГК РФ)

### 6. 🏪 Договор коммерческой концессии - Франчайзинг (franchise)
**Назначение:** Комплексная передача прав на использование объекта ИС в коммерческих целях  
**Ключевые разделы:**
- Предмет договора
- Предоставляемые права
- Территория и срок
- Вознаграждение
- Обязательства сторон (техподдержка, консультации)

### 7. 🔐 Договор залога исключительного права (pledge)
**Назначение:** Использование прав на объект ИС в качестве обеспечения обязательств  
**Ключевые разделы:**
- Предмет залога
- Обеспечиваемое обязательство
- Оценка предмета залога
- Права и обязанности залогодателя
- Права залогодержателя
- Требования к регистрации

### 8. 📢 Уведомление о нарушении (notice)
**Назначение:** Информирование нарушителя о факте нарушения с требованием устранения  
**Ключевые разделы:**
- Описание нарушения
- Правовые основания
- Требования (срок 10 рабочих дней)
- Предупреждение о судебных мерах

## Технические детали

### AI-генерация контента
Каждый тип документа использует специализированный AI-промпт для генерации юридически корректного контента на русском языке.

### Автоматическое обновление
После генерации документа список документов объекта автоматически обновляется без необходимости ручного обновления страницы.

### Шаблоны
Все шаблоны находятся в `backend/app/templates/` и используют формат Jinja2 для подстановки переменных.

### Fallback значения
Для каждого типа документа предусмотрены fallback значения на случай недоступности AI-сервиса.

## Использование

1. Выберите IP-объект
2. В разделе "Генерация документов" выберите тип документа
3. Выберите контрагента
4. Нажмите "Сгенерировать"
5. Документ автоматически скачается и появится в списке документов объекта
